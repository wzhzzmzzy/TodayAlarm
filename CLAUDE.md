# 定时提醒小助手

## 介绍

定时提醒小助手是一个基于 Android Compose 开发的定时提醒应用，它的主要能力是按照用户给定的计划，给用户设备发送系统级推送提醒。

用户可以自行设置计划，计划包括：提醒时间、提醒内容、是否重复提醒等。

提醒由本地应用即可发起，通过 AlarmManager + BroadcastReceiver 让系统到达指定时间时发起提醒通知。

## 架构设计

存储层使用简单的本地存储，存储计划信息。

业务层使用 MVVM 架构模式，将数据流和 UI 分离。

UI 层使用 Compose 开发，UI 层负责展示和交互。

## 交互设计

使用 Material 3 组件库开发 UI。

### 主页

主页标题为"提醒事项"，直接展示待办事项列表，让用户能够快速查看和管理当前的待办任务。

主页顶部导航栏右侧有两个按钮：
- 日历图标：点击跳转到周视图页面
- 列表图标：点击跳转到计划列表页面

主页右下角有浮动操作按钮，点击可以添加新计划。

### 添加计划

用户点击添加计划按钮，进入添加计划页面，可以设置提醒时间、提醒内容、是否重复提醒，添加计划后保存到本地。

### 周视图

用户点击主页顶部的日历图标，进入周视图页面。周视图展示本周的日历，从上往下排列，每一行是一天，高亮当前的这一天。日历上可以看到本周的待办事项和计划安排。

用户可以：
- 浏览不同周的日历
- 查看每天的计划和待办事项
- 点击日期查看该日的详细待办事项

### 计划列表

用户点击主页顶部的列表图标，进入计划列表页面，可以查看和编辑所有计划。

### 提醒通知

到达指定时间之后，系统会发起提醒通知，通知内容为计划内容，并且记录一个待办事项。

用户点击通知之后会进入应用，点击的这一条待办事项会以弹窗的形式出现，提醒用户点击完成，用户点击完成之后待办事项会被记录为已完成。

## 开发环境

- 语言：Kotlin
- 框架：Compose
- 存储：Room
